# 9/29 Score

It can print out the percentiles and top 5 scores and draw the histogram.

**Input:**

```
>> Score([80 72 61 55 43 58 90 78 32 10 50 36])
```

**Result:**

```
88th = 
    78

75th = 
    72

50th = 
    55

25th = 
    36

12th = 
    10

TOP 5 =
    61    72    78    80    90
```
